PROYECTO: Aplicación Web Responsive

¿Cómo ejecutar la página?:

Descargar o descomprimir la carpeta llamada xx_disWeb26.zip.

Abrir la carpeta xx_disWeb26.

Ejecutar el archivo index.html en cualquier navegador (Chrome, Edge o Firefox).

La página cargará el contenido dentro de un iframe, donde se mostrarán las diferentes secciones del proyecto.

No es necesario instalar nada adicional. Los audios y videos se cargan mediante enlaces externos desde la web.

Al iniciar la aplicación se muestra automáticamente un carrusel con 4 animales domésticos que se repite de forma infinita.

La barra de menú incluye tooltips en las opciones.

El mosaico de audios y videos está organizado utilizando CSS Grid.

El footer muestra a los integrantes del grupo

Finalmente aqui el despliegue en la page de git hub  https://meiweb24.github.io/05_disWeb26/
